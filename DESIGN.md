# 設計說明文檔

## 設計理念

本系統採用**極簡主義設計風格**，靈感來自歐美現代 Web 應用（如 Linear、Stripe、Vercel）。

### 核心原則

1. **Less is More**：只保留必要元素，去除裝飾性設計
2. **內容優先**：讓功能和內容成為焦點
3. **清晰層次**：通過留白和字體大小建立視覺層次
4. **一致性**：統一的間距、顏色、字體系統

---

## 配色方案

### 主色調
```css
--color-primary: #2563eb      /* 藍色 - 主要操作 */
--color-primary-hover: #1d4ed8 /* 藍色懸停 */
```

### 中性色
```css
--color-text: #0f172a         /* 深灰 - 主要文字 */
--color-text-light: #64748b   /* 淺灰 - 次要文字 */
--color-border: #e2e8f0       /* 邊框 */
--color-bg: #ffffff           /* 背景 */
--color-bg-light: #f8fafc     /* 淺背景 */
```

### 狀態色
```css
--color-success: #10b981      /* 綠色 - 成功 */
--color-error: #ef4444        /* 紅色 - 錯誤 */
```

### 為什麼選擇藍色？
- 專業、可信賴
- 不分性別，普遍接受度高
- 與黑白灰搭配和諧
- 符合歐美主流設計趨勢

## 字體系統

### 字體家族
```css
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 
             'Microsoft JhengHei', 'PingFang TC', sans-serif;
```
- 使用系統字體，加載快速
- 跨平台一致性
- 支持中英文混排

### 字體大小
```css
--font-size-3xl: 48px   /* 主標題 */
--font-size-2xl: 32px   /* 次標題 */
--font-size-xl: 24px    /* 小標題 */
--font-size-lg: 18px    /* 大正文 */
--font-size-base: 16px  /* 正文 */
--font-size-sm: 14px    /* 小字 */
```

### 字重
- **Regular (400)**：正文
- **Medium (500)**：強調
- **Semibold (600)**：小標題
- **Bold (700)**：主標題

### 行高
- 標題：1.2-1.4
- 正文：1.6-1.8（更易讀）

---

## 間距系統

採用 8px 基準的間距系統：

```css
--spacing-xs: 8px
--spacing-sm: 16px
--spacing-md: 24px
--spacing-lg: 32px
--spacing-xl: 48px
--spacing-2xl: 64px
```

### 使用原則
- 元素內部：xs, sm
- 元素之間：md, lg
- 區塊之間：xl, 2xl

## 組件設計

### 1. 按鈕

#### 主要按鈕（Primary）
- 藍色背景，白色文字
- 大尺寸：16px padding，18px 字體
- 懸停：顏色變深
- 禁用：50% 透明度

#### 次要按鈕（Secondary）
- 白色背景，灰色邊框
- 懸停：邊框變深

#### 圖標按鈕
- 無背景，只有圖標
- 用於輔助操作

### 2. 卡片

```
┌─────────────────────────┐
│  標題                   │
│  ─────────────────────  │
│  內容區域               │
│                         │
│  [操作按鈕]             │
└─────────────────────────┘
```

- 2px 邊框，12px 圓角
- 懸停：邊框變藍
- 內部間距：24px

### 3. 輸入框

- 2px 邊框，8px 圓角
- 焦點：藍色邊框
- 內部間距：12px

### 4. 下載卡片

每個語言一個卡片：
- 語言名稱（大字體）
- 格式選擇器（下拉菜單）
- 下載按鈕（全寬）

優點：
- 清晰的視覺分組
- 減少按鈕數量
- 用戶可以選擇格式

## 互動設計

### 1. 影片播放器

#### 字幕顯示
- 使用 HTML5 `<track>` 標籤
- 字幕疊加在影片上
- 半透明黑色背景
- 白色文字，易讀

#### 控制方式
- 點擊影片：暫停/播放
- 空格鍵：暫停/播放
- 原生控制條：音量、全螢幕、進度

### 2. 字幕編輯

#### 同步高亮
- 播放時自動高亮當前字幕
- 藍色左邊框 + 淺藍背景
- 自動滾動到當前位置

#### 編輯狀態
- 懸停：淺灰背景
- 編輯中：綠色左邊框
- 焦點：藍色邊框

#### 時間跳轉
- 點擊時間戳
- 影片跳轉到該位置
- 自動開始播放

### 3. 下載流程

```
選擇格式 → 點擊下載 → 瀏覽器下載
```

簡單直接，無需多步驟。

## 響應式設計

### 斷點
- 手機：< 768px
- 平板：768px - 1024px
- 桌面：> 1024px

### 適配策略

#### 手機
- 單列佈局
- 更大的觸控區域（48px 最小）
- 全寬按鈕
- 簡化導航

#### 平板
- 兩列佈局（下載卡片）
- 保持桌面體驗

#### 桌面
- 三列佈局（下載卡片）
- 最佳視覺效果

## 動畫與過渡

### 原則
- 快速（0.2s）
- 自然（ease）
- 不過度

### 使用場景
- 按鈕懸停
- 卡片懸停
- 邊框顏色變化
- 進度條填充

### 不使用動畫
- 頁面切換（太慢）
- 複雜動畫（分散注意力）
