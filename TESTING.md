# 功能測試清單

## 測試前準備
- [ ] 確認 START.bat 可以正常啟動服務器
- [ ] 確認瀏覽器可以訪問 http://localhost:8000
- [ ] 準備一個測試影片（建議 1-2 分鐘的短片）

## 基本功能測試

### 1. 上傳功能
- [ ] 點擊上傳區可以選擇文件
- [ ] 拖曳文件到上傳區可以上傳
- [ ] 選擇文件後顯示文件名
- [ ] 可以選擇目標翻譯語言（至少一種）
- [ ] 未選擇語言時提示錯誤
- [ ] 上傳成功後顯示進度條

### 2. 處理進度
- [ ] 進度條正常更新（0% → 100%）
- [ ] 顯示當前處理階段（轉錄、翻譯、生成）
- [ ] 處理完成後自動跳轉到結果頁面

### 3. 影片播放器 ⭐ 新功能
- [ ] 影片可以正常播放
- [ ] 播放/暫停按鈕正常工作
- [ ] 可以調整音量
- [ ] 可以全螢幕播放
- [ ] 可以拖動進度條

### 4. 字幕同步 ⭐ 新功能
- [ ] 播放影片時，當前字幕會自動高亮（黃色邊框）
- [ ] 字幕列表會自動滾動到當前播放位置
- [ ] 高亮效果流暢，沒有延遲

### 5. 字幕編輯 ⭐ 新功能
- [ ] 可以切換不同語言的字幕進行編輯
- [ ] 點擊字幕文字可以編輯
- [ ] 編輯時字幕框變成綠色邊框
- [ ] 編輯的內容即時更新
- [ ] 點擊「儲存修改」可以保存編輯
- [ ] 儲存成功後顯示成功提示
- [ ] 點擊「重置」可以恢復原始內容

### 6. 時間跳轉 ⭐ 新功能
- [ ] 點擊字幕的時間戳（例如：00:00:05.000 --> 00:00:08.000）
- [ ] 影片會跳轉到該時間點並開始播放
- [ ] 跳轉準確，沒有明顯偏差

### 7. 下載功能
- [ ] 每個語言都有 VTT 和 SRT 兩個下載按鈕 ⭐ 新功能
- [ ] 點擊 VTT 按鈕可以下載 VTT 格式字幕
- [ ] 點擊 SRT 按鈕可以下載 SRT 格式字幕 ⭐ 新功能
- [ ] 下載的文件名包含語言代碼
- [ ] 下載的字幕內容正確

### 8. 批量下載 ⭐ 新功能
- [ ] 點擊「批量下載所有字幕」按鈕
- [ ] 不勾選「包含影片檔案」，只下載字幕
- [ ] 下載的 ZIP 文件包含所有語言的字幕
- [ ] 勾選「包含影片檔案」，下載包含影片
- [ ] ZIP 文件名使用原始影片名稱

## 進階測試

### 9. 編輯後下載
- [ ] 編輯字幕內容
- [ ] 儲存修改
- [ ] 下載編輯後的字幕
- [ ] 打開下載的字幕文件，確認內容已更新

### 10. 多語言切換
- [ ] 在編輯器中切換不同語言
- [ ] 每個語言的字幕內容正確顯示
- [ ] 編輯一個語言不影響其他語言

### 11. 錯誤處理
- [ ] 上傳不支援的文件格式，顯示錯誤提示
- [ ] 上傳超過 5GB 的文件，顯示錯誤提示
- [ ] 未選擇語言就上傳，顯示錯誤提示

### 12. 響應式設計
- [ ] 在手機瀏覽器上測試（或縮小瀏覽器窗口）
- [ ] 影片播放器自動適應寬度
- [ ] 按鈕和控制項正常顯示
- [ ] 字幕編輯器可以正常使用

## 性能測試

### 13. 長影片測試
- [ ] 上傳 10-15 分鐘的影片
- [ ] 處理時間合理（約 5-10 分鐘）
- [ ] 字幕數量較多時（100+ 條），編輯器流暢
- [ ] 字幕同步沒有明顯延遲

### 14. 多語言測試
- [ ] 同時選擇 3 種語言翻譯
- [ ] 所有語言都正確生成
- [ ] 可以分別編輯每種語言

## 已知限制

1. 影片格式：只支持 MP4, AVI, MOV, MKV
2. 文件大小：最大 5GB
3. 影片時長：最長 2 小時
4. 翻譯語言：繁體中文、簡體中文、馬來文
5. 首次使用需要下載 Whisper 模型（約 139MB）
