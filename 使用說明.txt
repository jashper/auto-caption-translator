影片字幕自動翻譯系統 - 使用說明
=====================================

快速開始：
---------
1. 雙擊 START.bat 啟動服務器
2. 瀏覽器打開 http://localhost:8000
3. 上傳影片文件
4. 選擇需要翻譯的語言（繁體中文、簡體中文、馬來文）
5. 等待處理完成
6. 下載字幕文件

關閉服務器：
-----------
- 直接關閉服務器窗口
- 或在窗口中按 Ctrl+C

系統要求：
---------
- Python 3.10
- FFmpeg（已安裝）
- 至少 2GB 可用磁盤空間

支持格式：
---------
- 影片：.mp4, .avi, .mov, .mkv
- 最大文件：5GB
- 最長時長：2小時

語言選擇：
---------
- 系統會先將影片轉錄成英文字幕
- 您可以選擇需要翻譯的目標語言
- 至少選擇一種語言
- 只翻譯您選擇的語言，節省處理時間和 API 使用量

注意事項：
---------
- 首次處理會下載 Whisper 模型（約 139MB）
- 處理時間取決於影片長度
- 所有處理都在本地完成，不上傳到雲端
- 選擇性翻譯可以避免過度使用 Google Translate API
